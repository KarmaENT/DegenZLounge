DeGeNz Lounge - UI Mockups

The following mockups demonstrate the key features of the DeGeNz Lounge webapp with multi-model AI integration.

1. Main Interface

The main interface is divided into three panels:

  • Left: Agent library with draggable agent cards
  • Center: Sandbox environment with chat interface
  • Right: AI model settings and configuration

Multi-Model Integration

The interface allows users to select from multiple AI providers including Gemini (default), DeepSeek, Grok, Hugging Face, OpenRouter, Anthropic, Mistral AI, Perplexity, and Ollama (self-hosted).

2. Agent Library

The Agent Library panel features:

  • Search functionality for finding specific agents
  • Agent cards with role, personality, and specialization
  • Visual indicators showing which AI model powers each agent
  • Drag-and-drop functionality for adding agents to the sandbox
  • "Create New Agent" button for defining custom agents

3. Sandbox Environment

The Sandbox Environment includes:

  • Dropzone for adding agents from the library
  • Active agent indicators showing which agents are in the workspace
  • Discord-style threaded chat interface
  • Manager agent that orchestrates tasks between agents
  • Message input with configurable collaboration modes

Collaboration Modes

The Manager Agent can operate in two modes:

  • Strict: Manager tightly controls the workflow and assigns specific tasks
  • Collaborative: Agents have more autonomy to interact and contribute

4. AI Model Settings

The AI Model Settings panel provides:

  • Provider selection with 9 different AI platforms
  • Model selection specific to each provider
  • Option to show only free tier models
  • Usage statistics and quota monitoring
  • Free credits display showing available resources
  • Usage optimization tools to maximize free tier benefits

5. Agent Creation Interface

The Agent Creation interface allows users to define custom agents with:

  • Role definition (e.g., Researcher, Copywriter, Designer)
  • Personality traits to shape agent responses
  • Specialization areas for focused expertise
  • System instructions for behavior guidelines
  • Example interactions for training
  • AI model selection for powering the agent

Free Tier Optimization

The application includes features to maximize the use of free API tiers:

  • Automatic switching between providers based on quota availability
  • Response caching to reduce duplicate API calls
  • Token usage tracking across different providers
  • Optimization suggestions for efficient prompt engineering

6. Example Workflow

This mockup demonstrates a complete workflow:

  1. User inputs a task: "Plan eco-friendly shoe launch"
  2. Manager Agent assigns subtasks to specialized agents
  3. Researcher Agent provides market analysis (powered by Gemini)
  4. Copywriter Agent creates slogans (powered by Mistral)
  5. Designer Agent develops packaging concepts (powered by DeepSeek)
  6. All agents collaborate with shared context and memory
